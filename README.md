# Muhembo Bus Reservation System

A simple React bus reservation system for Muhembo Bus.

## Features

- Passenger sign up and login
- Forgot password/reset password demo
- Admin login
- Route viewing and search
- From town / To town selection
- Seat selection layout
- M-Pesa, card, and cash payment demo
- My bookings page
- Admin dashboard with routes, bookings, and revenue
- Add routes and delete bookings

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this project to GitHub.
2. Go to Vercel and click **Add New Project**.
3. Import your GitHub repository.
4. Framework preset: **Vite**.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

## Note

This version stores user accounts in browser localStorage for demo purposes. For a real production system, connect it to a backend database such as Supabase, Firebase, or MySQL.
